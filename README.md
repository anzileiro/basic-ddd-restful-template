# get-tweets-cron

## Build
```terminal
docker build --tag zultest:1.0 .  
```

## Run
```terminal
docker run zultest:1.0   
```

## Installing dependencies
```terminal
npm install
```

## Unit Tests
```terminal
npm run test
```

## Tests Coverage Report
```terminal
npm run coverage
```
